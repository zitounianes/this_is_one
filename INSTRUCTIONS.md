# تعليمات تشغيل مشروع المطعم

تم تحويل المشروع ليعمل بالكامل باستخدام Next.js مع Backend مدمج.

## المتطلبات
- Node.js مثبت على جهازك.

## خطوات التشغيل

1. **تثبيت المكتبات:**
   افتح موجه الأوامر (Terminal) في مجلد المشروع ونفذ الأمر التالي:
   ```bash
   npm install
   ```

2. **إعداد قاعدة البيانات:**
   لإنشاء ملف قاعدة البيانات (SQLite) وتطبيق الهيكلية، نفذ:
   ```bash
   npx prisma db push
   ```
   *(سيؤدي هذا إلى إنشاء ملف `dev.db` في مجلد `prisma`)*.

3. **تشغيل السيرفر:**
   لبدء تشغيل التطبيق في وضع التطوير:
   ```bash
   npm run dev
   ```

4. **فتح التطبيق:**
   الآن يمكنك فتح الصفحات التالية في المتصفح:
   - الصفحة الرئيسية: [http://localhost:3000/index.html](http://localhost:3000/index.html)
   - لوحة التحكم: [http://localhost:3000/admin.html](http://localhost:3000/admin.html)

## ملاحظات هامة
- جميع بياناتك (الوجبات، الطلبات، الإعدادات) سيتم حفظها في قاعدة البيانات المحلية (`prisma/dev.db`).
- تم نقل ملفات الواجهة (HTML, CSS, JS) إلى مجلد `public` ليتم خدمتها بشكل مباشر.
- الـ API يعمل الآن تحت المسار `/api` (مثل `/api/meals`, `/api/orders`).
